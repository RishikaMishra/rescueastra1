### Project Overview: RescueAstra

**RescueAstra** is a cross-platform emergency response and safety management application, designed to help users report incidents, analyze hotspots, and access emergency services efficiently.

#### Key Features & Implementations

- **Incident Reporting & Management**
  - Users can report incidents with details such as location, type, severity, description, and additional metadata.
  - Each incident can include tags, detected threats, gender information, and features (e.g., crowd, vehicle).
  - Incidents are timestamped and geolocated for accurate tracking.

- **Hotspot Analysis**
  - The system aggregates incident data to identify and analyze high-risk areas (“hotspots”).
  - Hotspot models include risk scores, incident counts, risk levels, common tags, and time-based patterns.
  - Provides summaries of detected threats, gender distribution, and other features within hotspots.

- **Emergency Services Integration**
  - Quick access to emergency contacts and services from within the app.
  - Features for adding and managing personal emergency contacts.

- **User Authentication & Profile Management**
  - Supports user sign-up, login, and password recovery.
  - User profiles are managed for personalized experience and incident attribution.

- **Interactive Map & Location Search**
  - Visualizes incidents and hotspots on an interactive map.
  - Allows users to search for locations and view risk information.

- **Cross-Platform Support**
  - Built with Flutter, supporting Android, iOS, web, Windows, Linux, and macOS platforms.

- **Firebase Integration**
  - Uses Firebase Firestore for real-time data storage and retrieval.
  - Leverages Firebase for authentication and cloud functions.

- **Extensible Data Models**
  - Models are designed to accommodate future enhancements, such as advanced threat/gender detection, additional incident features, and analytics.

- **User-Friendly Interface**
  - Clean, intuitive UI for reporting incidents, viewing hotspots, and accessing emergency features.
  - Includes chatbot support for user assistance.

---

**Note:**  
While the codebase is structured to store results of gender and threat detection, the actual detection algorithms (e.g., AI/ML models) are not implemented within this repository. The system is designed to integrate with such services if needed.
